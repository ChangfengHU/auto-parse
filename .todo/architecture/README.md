## 技术选型

| 层级 | 技术 | 理由 |
|------|------|------|
| 前端 | Next.js + React | 现有技术栈 |
| 后端 | Next.js API Routes | 全栈统一 |
| 自动化 | Playwright | 功能完整、社区活跃 |
| 数据库 | Supabase (PostgreSQL) | 免运维、实时订阅 |
| 对象存储 | 阿里云 OSS | 国内访问快 |
| 客户端 | Tauri + Rust | 轻量、安全 |
