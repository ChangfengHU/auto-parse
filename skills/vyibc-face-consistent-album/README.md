# vyibc-face-consistent-album

全感官影像艺术生成器 (V4.5 Pro Max)。基于“全场景通用美学”逻辑，提供闭环识图与多维感官控制。

---

## 📖 使用方式

用户只需提供：
- **参考图 URL**：唯一身份资产。
- **主题场景**：不限场景（办公室、废墟、书店、街头等）。
- **美学风格**（可选）：
  - `master_allure`：全域魅惑，强调眼神拉扯、肢体张力与材质触感（高级性感）。
  - `pure_ethereal`：白幼瘦/纯欲，强调瓷感肤色、无辜神态与纤细灵动。
  - `fuji`：富士大师，强调抓拍感与复古影调。

## 🎯 触发规则

当用户表达以下意图或关键词时自动触发：
- "用 [URL] 创作一组 [主题] 大片，风格要 [风格名]"
- "想要一套吸睛的 [白幼瘦/性感] 风格图"
- 关键词：`全感官`、`双美学`、`全域魅惑`、`白幼瘦`、`极致影像`

## 💎 终极控制标准 (V4.5 Standards)

每一张影像的生成都会经过以下四个感官维度的洗礼：

1. **灵魂神态 (Soul & Gaze)**：捕捉故事感眼神与微表情。
2. **人体韵律 (Body Flow)**：利用光影勾勒皮肤线条与动态美感。
3. **材质交互 (Material & Tactility)**：定义衣物与皮肤接触的真实物理感。
4. **大师构图 (Master Cinematography)**：利用极端视角营造吸睛视觉深度。
