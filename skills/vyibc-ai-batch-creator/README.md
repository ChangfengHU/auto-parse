# vyibc-ai-batch-creator

🎨 **AI 创意批量生成器** —— 基于参考图快速生成多张风格一致的创意图片

## ⚡ 核心能力

| 能力 | 说明 |
|------|------|
| **批量生成** | 一次生成 5-100+ 张，无需逐个等待 |
| **一致性** | 自动保持参考图的风格和人物特征 |
| **大规模** | 支持 50+ 张的大规模创作 |
| **实时监控** | 支持进度查询和实时反馈 |

## 🚀 30 秒快速开始

```bash
/vyibc-ai-batch-creator
  sourceImageUrl: "https://example.com/girl.jpg"
  prompts:
    - "女生在沙滩，白色连衣裙，笑容灿烂"
    - "女生在办公室，职业套装，专业气质"
    - "女生在家里，家居服，放松表情"
```

✅ **结果**：3 张图片，同一个人在不同场景

### 参数速查

| 参数 | 类型 | 必须 | 说明 |
|------|------|------|------|
| `sourceImageUrl` | string\|string[] | ❌ | 参考图 URL |
| `prompts` | string[] | ✅ | 创意描述列表 |
| `priority` | boolean | ❌ | 优先执行（跳过队列）|
| `maxAttempts` | number | ❌ | 重试次数（默认 3） |
| `timeout` | number | ❌ | 超时时间（毫秒） |

### 输出信息

## ❓ 常见问题

**Q: 支持多少张?**
A: 5-200 张都可以

**Q: 要多久?**
A: 通常 3-15 分钟

**Q: 参考图重要吗?**
A: 很重要！高质量的参考图能更好地保持一致性

**Q: 生成失败了?**
A: 系统会自动重试，可通过 `maxAttempts` 调整重试次数

**Q: 如何查看进度?**
A: 使用返回的 `queryUrl` 或在浏览器访问

**Q: 可以取消吗?**
A: 可以，使用返回的 `cancelUrl`

**Q: 支持哪些场景?**
A: 任何需要批量生成一致风格图片的场景
- 个人写真
- 故事配图
- 产品展示
- 角色原画
- 同人创作
- 品牌资产
- ...你的想象力是极限

## 🎯 最佳实践

### 1️⃣ 准备好参考图
- 使用清晰、高质量的参考图
- 可以是 1-3 张
- URL 需要可访问

### 2️⃣ 编写详细的 Prompts
```
✅ 好的 Prompt:
"女生在海边，穿着白色连衣裙，手拿冰淇淋，阳光灿烂，笑容满面"

❌ 不好的 Prompt:
"女生在海边"
```

### 3️⃣ 保持 Prompt 风格一致
```
✅ 一致的描述格式:
- "角色在[场景]，穿着[服装]，[表情/气质]"
- "角色在[场景]，穿着[服装]，[表情/气质]"
- ...

❌ 不一致:
- "角色在海边..."
- "画一个女生在..."
- "需要一张..."
```

### 4️⃣ 使用优先级（如果急需）
```bash
priority: true   # 跳过队列，立即执行
```
