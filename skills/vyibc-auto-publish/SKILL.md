## 使用方式

调用 `scripts/publish.sh` 脚本完成发布：

```bash
bash scripts/publish.sh "<ossUrl>" "<标题>" ["话题1,话题2"]
```

## 触发规则

用户提供以下内容时触发：
- 说"发布到抖音"、"上传到我的抖音"、"帮我发到抖音"
- 已有 `ossUrl`（来自解析结果）且明确要发布
- 先解析视频，再一键发布的连续流程

## 执行步骤

1. 从上下文或用户输入中获取 `ossUrl`（OSS视频地址）
2. 确认发布标题（可用视频原始文案的前30字，或让用户指定）
3. 运行脚本：`bash scripts/publish.sh "<ossUrl>" "<title>" ["tag1,tag2"]`
4. 实时展示流式进度日志给用户
5. 若出现 `[QRCODE]` 行，告知用户扫码（此时页面会显示二维码图片）
6. 等待 `[DONE]` 或 `[ERROR]` 结束

## 错误处理

| 错误信息 | 处理方式 |
|---------|---------|
| 当前有发布任务正在进行中 | 等上一个任务完成再重试 |
| Cookie 过期 | 提示用户在网页端扫码（`http://localhost:1007`）|
| 视频上传失败 | 检查 OSS 地址是否有效，重试 |
| 内容违规/检测不通过 | 告知用户视频内容可能违规，需更换视频 |
| 标题填写失败 | 检查标题是否包含特殊字符，尝试简化标题 |
